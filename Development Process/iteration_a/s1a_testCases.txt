Test case file for s1a which takes in user input and stores in 9 3x3 2d 
matrices called grids1, grids2, ..., grids9 which are stored in a 3x3 cell
called grids.


User inputs are separated by a space, the program asks for 81 number 
inputs, which it sorts into grids

0 6 0 8 0 3 0 2 0        grids{5} = [0 6 0; 8 0 3; 0 2 0];
